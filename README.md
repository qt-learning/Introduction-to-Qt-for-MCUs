# Getting Started with Qt for MCUs

This is the example project used in the "Getting Started with Qt for MCUs" course.

> Get the full learning experience by enrolling in the course on [Qt Academy](https://www.qt.io/academy/course-catalog#getting-started:-qt-for-mcus)

## 🎯 What You'll Learn

In this course, you'll get an overview of what Qt for MCUs is, and why use it. You will create your first project with Qt for MCUs: a simple heartbeat monitor. We will create a graphic user interface, build a backend, and deploy it to a target.

This course is for people interested in developing on microcontrollers (MCUs). We recommend that you have previous knowledge of Qt as we'll be using Qt Design Studio and Qt Creator.

⭐ Skill Level: Beginner

⏱️ Time to Complete: 30 minutes

# Getting Started

To get started with this project, you will need to have Qt for MCU, Qt Creator, and Qt Design Studio installed on your computer. You can download them from the Qt.io website.

Once you have Qt for MCU, Qt Creator, and Qt Design Studio installed, head over to Qt Academy and follow along to create the project template and add the assets to the project.

You can download the project by selecting the green button "Code" -> Download ZIP.

After downloading, extract the zip file to your computer and open the `HeartbeatMonitor.qmlproject` file in Qt Design Studio. To open the project in Qt Creator, open the `CMakeLists.txt` file.
